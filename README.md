# Olympic-Analyzer
An simple olympic analyzer from an external data source.

<img src="Olympic Analyzer -Thumbnail.png">

## How it works?
Checkout my youtube video-> https://youtu.be/J0UxjqieUvc

## Try Demo!
No need to install try it out -> https://olympicanalyzers.herokuapp.com/

## Requirements
- matplotlib==3.5.2
- numpy==1.22.4
- pandas==1.4.1
- plotly==5.9.0
- seaborn==0.11.2
- streamlit==1.10.0

## Installation

Must have to satisfy all the requirements

```bash
  streamlit run app.py
```

## FAQ

#### When to use it?
If you're an olympic lover and wanna know full history and analysis of olympic then this tool is for you.
